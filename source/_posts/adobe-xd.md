## Keyboard Shortcuts

### Keys for Edit menu

| Shortcut           | Action                                                               |
| ------------------ | -------------------------------------------------------------------- |
| `Ctrl` `Z`         | Undo                                                                 |
| `Ctrl` `Shift` `Z` | Redo                                                                 |
| `Ctrl` `X`         | Cut                                                                  |
| `Ctrl` `C`         | Copy                                                                 |
| `Ctrl` `V`         | Paste                                                                |
| `Ctrl` `Alt` `V`   | Paste Appearance (Design mode) or Paste Interaction (Prototype mode) |
| `Ctrl` `D`         | Duplicate                                                            |
| `Del`              | Delete                                                               |
| `Ctrl` `A`         | Select All                                                           |
| `Ctrl` `Shift` `A` | Deselect All                                                         |
