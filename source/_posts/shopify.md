## Keyboard Shortcuts

### General Shortcuts

| Shortcut    | Action                  |
| ----------- | ----------------------- |
| `?`         | Keyboard help           |
| `S`         | Focus search bar        |
| `F`         | Focus filter bar        |
| `B` `Y` `E` | Log out                 |
| `M` `E`     | Your profile            |
| `O` `S`     | Open the store switcher |
