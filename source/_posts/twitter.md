## Keyboard Shortcuts

### Actions

| Shortcut       | Action             |
| -------------- | ------------------ |
| `N`            | New tweet          |
| `L`            | Like tweet         |
| `R`            | Reply to tweet     |
| `T`            | Retweet            |
| `M`            | Direct message     |
| `U`            | Mute account       |
| `B`            | Block account      |
| `Enter`        | Open tweet details |
| `O`            | Expand photos      |
| `/`            | Search             |
| `Ctrl` `Enter` | Send tweet         |

{.shortcuts}

### Navigation

| Shortcut | Action             |
| -------- | ------------------ |
| `?`      | Full keyboard menu |
| `J`      | Next tweet         |
| `K`      | Previous tweet     |
| `Space`  | Page down          |
| `.`      | Load new tweets    |

{.shortcuts}
